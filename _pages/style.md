## Editorial

* Write using [plain language](https://www.plainlanguage.gov/guidelines/)
* Follow 18Fâ€™s [inclusive language](https://content-guide.18f.gov/our-style/inclusive-language/) style guide
* Write for a non-technical audience.
* Be concise
* Avoid acronyms or jargon
* Use active voice instead of passive voice
* Use descriptive links
* Avoid using footnotes
